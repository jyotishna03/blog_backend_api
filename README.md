📰 Blog Backend API (Next.js + MongoDB)

A simple and efficient Blog Backend API built with Next.js 14 (App Router) and MongoDB.
This backend enables users to create, read, update, and delete blog posts seamlessly using RESTful endpoints.

🚀 Tech Stack

Next.js 14 (App Router) — API routing and serverless functions

MongoDB + Mongoose — Database and schema management

TypeScript — Type-safe and scalable code

Node.js — Runtime environment

📦 Features

✅ Full CRUD operations
✅ Schema-defined posts (title, content, author, date)
✅ Clean, modular structure
✅ MongoDB connection via .env.local
✅ Easy integration with frontend apps
